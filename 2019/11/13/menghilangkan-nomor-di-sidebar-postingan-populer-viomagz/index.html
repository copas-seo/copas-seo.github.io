Araz Seo : Di kesempatan ini saya ingin share bagaimana Cara ter-mudah menghilangkan nomor yang ada di sidebar postingan populer pada template Viomagz terbaru…

>>>>>>> 16af3e9eba0d30886779ea6cccef3921e510fb07

Perlu di ingat sebelum Anda ingin merubah atau mengedit template melalui HTML biasakan back-up terlebih dahulu templatenya, jadi saat terjadi kesalahan edit Anda masih memiliki file HTML aslimya yang bisa kita restore.

  1. Masuk ke akun Blogger anda.
  2. Pilih Menu Tema yang ada di kiri.
  3. Lalu klik Edit HTML

Kemudia Anda Cari kode seperti di bawah ini, agar tidak kesulitan gunakan combinasi keyboard dengan cara menekan CTRL + F

<b:include name=’numberedpopularposts’/>

Jika sudah ketemu silahkan di hapus tersebut dan klik simpan Tema, lalu lihat hasilnya, Doneee Anda sudah berhasil menghilangkan nomor yang ada di postingan populer di widget template Viomagz.

Memang ada banyak tutorial Cara Menghlangkan Nomor yang ada di sidebar postingan populer Viomagz ini,  namun agak sulit untuk menerapkannya, apalagi untuk pemula blogger yang baru memakai template Viomagz dan  belum banyak tau mengenai HTML pasti sangat membingungkan.
